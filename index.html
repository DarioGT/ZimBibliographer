Zim is a desktop wiki. ZimBibliographer is a command line tool using a bibtex to add references.

In your notes, you can write keys like
cite{Author1976}

ZimBibliographer will replace that by a link to the file (pdf, ps...) included your bibtex you manage elsewhere.

Usage¶

zimbibliographer.py -h
